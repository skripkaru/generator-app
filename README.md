# Generator App

## Demo

<https://generator-app-five.vercel.app/>

## Tech Stack

**Client:** Next.js, Typescript, Ant Design

## Installation

```bash
  yarn install
```

### Starts in development mode with hot reload

```bash
  yarn dev
```

### Starts in production mode

```bash
  yarn start
```

### Lints all files

```bash
  yarn lint
```

### Creates an optimized production build

```bash
  yarn build
```
